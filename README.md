# Market Segmentation Using Machine Learning Algorithms

Welcome to the **Market Segmentation Using Machine Learning Algorithms** repository. This project implements a professional, end-to-end unsupervised data science pipeline designed to discover, profile, and evaluate distinct customer segments from large-scale transactional data.

By engineering behavioral features and leveraging advanced clustering techniques, this system transforms raw historical logs into highly actionable business cohorts, enabling data-driven marketing, customer relationship management (CRM), and lifetime value (LTV) optimization.

---

## 🚀 Features & Pipeline Architecture

The repository contains a fully documented, structured Jupyter Notebook pipeline including:

1. **Environment Setup:** Production-ready imports utilizing `pandas`, `numpy`, `scikit-learn`, `matplotlib`, and `seaborn`.
2. **Data Cleansing & Preprocessing:** * Dynamic handling of missing identity records (`CustomerID`).
   * Identification and removal of cancellation records (invoices starting with `C`).
   * Statistical treatment for outliers and negative quantity/price artifacts.
3. **RFM Feature Engineering:** Compiling granular transaction data into customer-level metrics:
   * **Recency ($R$):** Days since the customer's last purchase.
   * **Frequency ($F$):** Total number of distinct transactions.
   * **Monetary ($M$):** Net financial value generated by the user.
4. **Data Normalization:** Standardization via `StandardScaler` to ensure scale-invariant distance calculations.
5. **Dimensionality Reduction:** Geometric simplification and variance retention using Principal Component Analysis (PCA).
6. **Algorithmic Clustering & Optimization:**
   * **K-Means Clustering:** Optimized using the Elbow Method (WCSS), Silhouette Analysis, and the Davies-Bouldin Index.
   * **Hierarchical Agglomerative Clustering:** Structured using Ward's Linkage and validated via Dendrogram visualization.
7. **Cluster Profiling:** Comprehensive statistical and business mapping of the resulting segments.
